/**
 * This is the descriptive information in regards to the settings file
 */

/**
 * Storage
 */
 The settings configuration file is stored in ./cfg/settings.cfg
 
/**
 * Layout
 */
 The contents of this file will be set-up in the methodology of:
_KEY_=_VALUE_

/**
 * Structure
 */
_KEY_ indicates the settings name that will be modified

_VALUE_ indicates the value to assign to the given _KEY_

The individual parameters are separated by an equals sign (=)
